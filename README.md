# 📈 Smart Real-Time Stock Dashboard

A professional stock market dashboard built using Python, Streamlit, Plotly, and yFinance.

---

## 🚀 Features

- 📊 Live stock market data
- 📈 Candlestick charts
- 🤖 AI Buy/Sell signals
- 📉 RSI Indicator
- 📦 Trading volume analysis
- 🏢 Company information
- 📰 Latest stock news
- 💰 Portfolio estimator
- 📥 Download CSV data

---

## 🛠 Technologies Used

- Python
- Streamlit
- Plotly
- yFinance
- Pandas
- NumPy

---

## 📂 Project Structure

```bash
stock-dashboard/
│
├── app.py
├── README.md
├── requirements.txt
```

---

## ▶️ Installation

Install required packages:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python -m streamlit run app.py
```
